使用教程

{{ site_name }} — 快速上手，3 分钟学会使用知识库

一、快速开始

1
创建知识库 登录后进入「我的知识库」→ 点击「新建知识库」，填写名称和描述。
2
上传文档 进入知识库仪表盘 → 「更多操作」→「上传文档」，支持 Markdown、PDF、Word、Excel、CSV 等格式。上传后系统会自动解析并生成 Wiki 页面。
3
等待摄入完成 上传后点击「开始摄入」，系统后台会解析文档、提取知识、建立索引。进度条显示处理进度。
4
提问查询 在知识库页面底部的输入框中输入问题，系统会基于所有已摄入文档进行检索并生成回答。

二、回答解读

每个回答包含以下信息：

  • 正文回答 — AI 根据知识库内容生成的回答
  • 置信度 — 高 / 中 / 低，反映检索证据的充分程度
  • 查看溯源 — 点击可查看回答引用的原始来源（Wiki 页面、文档片段、结构化数据）
  • 查询 ID — 每次查询的唯一标识，如果回答有问题可以将 ID 反馈给管理员
关于置信度：低置信度意味着知识库中匹配的内容较少，回答可能不够准确。建议在低置信度时查看溯源信息确认来源。

三、回答反馈

每条回答底部有 👍 / 👎 按钮：

  • 如果回答准确有用，点 👍
  • 如果回答不准确或有遗漏，点 👎，可补充简要说明

您的反馈将帮助管理员持续改进知识库质量。

四、支持的文档格式

格式说明
Markdown (.md)直接作为 Wiki 页面
PDF通过 MinerU 解析提取文字
Word (.docx)自动转为 Markdown
Excel (.xlsx / .xls)转为 Markdown 表格 + 结构化事实索引，保留原始文件
CSV同 Excel 处理方式

五、常见问题

Q：上传后没有 Wiki 页面？
请检查摄入任务是否完成。进入知识库仪表盘查看任务状态。
Q：回答说"无法回答"？
可能知识库中没有相关内容。请确认文档已成功摄入，且问题与文档内容相关。
Q：回答不完整或被截断？
遇到回答异常，请点 👎 反馈并附上查询 ID，管理员会排查。
Q：Excel 中的数据查不到？
Excel 会自动建立结构化索引，支持按字段精确查询（如"LLaMA 2 的训练数据量"）。如仍查不到，请联系管理员重建索引。
