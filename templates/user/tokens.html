使用方法

所有 API 端点位于 /api/v1/**，使用 Bearer Token 鉴权。

# 1. 验证 token / 查看账号信息
curl -H "Authorization: Bearer <your-token>" \
  https://your-domain/api/v1/me

# 2. 列出当前 token 可见的全部知识库
curl -H "Authorization: Bearer <your-token>" \
  https://your-domain/api/v1/repos

# 3. 在指定知识库检索（显式路由）
curl -X POST https://your-domain/api/v1/search \
  -H "Authorization: Bearer <your-token>" \
  -H "Content-Type: application/json" \
  -d '{"query":"AE350 的接口参数", "repo":"alice/ae350-kb"}'

# 4. 自然语言自动选库（由后端 LLM 选择最匹配的知识库）
#    成功时返回检索结果，routing.mode=auto；
#    置信度不足或无匹配时返回 422 + candidates 候选列表。
curl -X POST https://your-domain/api/v1/search \
  -H "Authorization: Bearer <your-token>" \
  -H "Content-Type: application/json" \
  -d '{"query":"帮我查 AE350 的接口参数"}'
